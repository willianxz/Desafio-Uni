Primeiramente antes de tudo, rode os serviços de banco de dados (Mysql) com (Xamp) ou qualquer outro, em seguida crie o banco de dados com o Mysql Workbeach usando o arquivo "modelobanco.mwb" abrindo ele, selecionando então "Database" no menu em cima, depois vá em Syncronize Model dando next, até ser criado o banco de dados. Caso não tenha o Mysql Workbeach tente usar o arquivo sqlcreatedb.sql, que contem a sintaxy sql para criar o banco de dados.

Após ter o banco de dados criado, você pode usar o arquivo insertsbanco.sql que já contem alguns inserts prontos para serem executados, fornecendo assim dados iniciais para o sistema.

Na pasta config, há um arquivo chamado "conectdb.php", caso você não esteja usando o localhost ou esteja usando alguma senha no seu banco, abra esse arquivo e preencha a conexão conforme você esteja utilizando.
